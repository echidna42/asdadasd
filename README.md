# Training Planner

Автор: [Ваше имя и фамилия]

Описание:
Это простое GUI-приложение для планирования тренировок. Позволяет добавлять тренировки с указанием даты, типа и длительности, фильтровать записи по типу и дате, а также сохранять данные в JSON-файл.

Использование:
1. Введите дату в формате YYYY-MM-DD.
2. Укажите тип тренировки.
3. Укажите длительность в минутах.
4. Нажмите «Добавить тренировку».
5. Для фильтрации введите критерии в соответствующие поля и таблица обновится автоматически.

Данные сохраняются при закрытии приложения и загружаются при запуске.
